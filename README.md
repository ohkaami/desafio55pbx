Automação de teste simples com Robot Framework.

* Abrir o navegador no site escolhido
* Escrever o produto na barra de pesquisa ("X Box")
* Verificar se a pesquisa foi bem sucedida
* Clicar no elemento correspondente
* Adicionar ao carrinho
